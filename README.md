# Tugas Rekursif – Struktur Data

## Daftar Tugas

| No | Tema | File | Deskripsi |
|----|------|------|------------|
| 1 | N-Queens | `1_n_queens.py` | Menempatkan N ratu di papan N×N tanpa saling menyerang |
| 2 | Knight's Tour | `2_knights_tour.py` | Tur kuda yang mengunjungi semua petak tepat satu kali |

## Algoritma yang Digunakan

- **Backtracking Rekursif** – untuk eksplorasi semua kemungkinan
- **Warnsdorff's Heuristic** – untuk optimasi Knight's Tour
- **Branch & Bound** – untuk efisiensi Knapsack
